GeoRef
======

Le module GeoRef est en charge de gérer le référentiel des données géographiques.

Ce référentiel peut être alimenté par différentes sources de données telles que :

* Téléatlas
* OpenStretMap
* ...

----------------------

ce module est en charge de :

* Gérer le géocodage.

-----------------------------

* Une adresse
